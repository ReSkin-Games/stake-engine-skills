# Setup

Install the Stake Engine Math SDK and prepare a Python environment to run simulations.

Upstream repo: `https://github.com/StakeEngine/math-sdk`.

## Prerequisites

- Python 3.12+
- Rust/Cargo (only required if the Rust-backed optimization algorithm is used)
- POSIX shell (`make` targets assume bash/zsh)

## Clone and install

```bash
git clone git@github.com:StakeEngine/math-sdk.git
cd math-sdk
make setup
```

`make setup` provisions a Python virtual environment and installs the Math SDK's Python dependencies into it. After it completes, the venv is the canonical interpreter for any `make run` and any direct invocation of `games/<game>/run.py`.

## Run a sample game

```bash
make run GAME=0_0_lines
```

`GAME` is the folder name under `math-sdk/games/`. Outputs land in `math-sdk/games/<GAME>/library/`.

## Manual invocation

After the venv is activated, a game may be run directly:

```bash
python3 games/0_0_lines/run.py
```

## Directory created on first run

`library/` and its subfolders (`books/`, `books_compressed/`, `configs/`, `forces/`, `lookup_tables/`, `publish_files/`) are auto-generated by `create_books()` and `generate_configs()` if absent.
